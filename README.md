# beton

Public metadata repository for the Beton Solana program.

The Beton on-chain program code is closed source and is not published in this repository.

This repository exists only to host the public metadata referenced by Beton's on-chain `security.txt`.

Contents:

- `README.md` — repository scope and disclosure posture
- `SECURITY.md` — vulnerability reporting and operational security policy

For the live application, see https://betonbtc.online.
